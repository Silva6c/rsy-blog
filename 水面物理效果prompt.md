# 水面物理效果 — 完整实现规格

## 场景

正对一方形玻璃水缸。视线**垂直**穿过水壁。屏幕上半是空气，下半是水。缸有前后左右底五面玻璃壁。

## 水面 = 三维高度场 h(x, z)

水面是定义在水平面 X-Z 上的三维曲面：

| 轴 | 方向 | 说明 |
|----|------|------|
| **X** | 左右 | 屏幕宽度方向，波的主传播轴 |
| **Z** | 纵深 | 垂直屏幕指向远方，z=0 为前壁，z=z_max 为后壁。不宜过长 |
| **Y** | 垂直 | 水面高度。某点 h(x,z) 大 = 水深 = 水面线在屏幕上位置靠上 |

## 波传播

涟漪沿 X 和 Z 两个方向传播。二维波方程 `∂²h/∂t² = c²(∂²h/∂x² + ∂²h/∂z²)`，阻尼系数衰减。侧壁和后壁反射波叠加。

## 水面运动有两种

### 1. Z方向整体倾斜

水面可以前低后高（`h(x, z_max) > h(x, 0)`）或前高后低。鼠标垂直移动驱动。

### 2. 表面波

涟漪在 X-Z 平面传播，任意位置有局部隆起凹陷。鼠标划过产生涟漪脉冲。

## 视线与水面的关系

视线垂直正对水壁。水面完全水平时 → 法线与视线重合 → 入射角零 → 无折射。折射只在界面法线偏离视线方向时发生。

## 界面的法线由三维梯度决定

法线 = `(-∂h/∂x, 1, -∂h/∂z)` 归一化。

- ∂h/∂x 贡献折射的**水平偏转**
- ∂h/∂z 贡献折射的**纵深偏转**

## 你看到的

**水面前缘**：z=0 处水与玻璃前壁的交线，Y 值 = h(x, 0)。沿 X 方向波动。

**上半屏（空气层）**：正常背景。

**下半屏（水体层）**：水下背景。折射扭曲仅发生在界面处——光线从空气入水时弯折一次，水体内部静止均匀，不再叠加扭曲。

## 折射的三种效果

### 全局水平平移
X 方向整体倾斜（如左高右低）→ 法线向一侧偏 → 下半屏图像整体水平偏移。

### Z倾斜的渐变偏转
水面前低后高 → 远端和近端折射量不同 → 体量感。

### 局部透镜扭曲
波峰像凸透镜→局部放大/挤压。波谷像凹透镜→反向扭曲。靠近水面线处最明显。

## 驱动与衰减

- **鼠标水平移动** → X 方向倾斜 + 涟漪
- **鼠标垂直移动** → Z 方向倾斜
- **鼠标静止** → 阻尼弹簧回归水平，波衰减至零

## 水的颜色

物理上水透明无色。能否感知水的存在取决于：水面线起伏 + 背景折射扭曲。仅叠 10%~20% 淡蓝为视觉提示。

## 边界条件

- **前壁** z=0：水面线与空气交界的切面
- **后壁** z=z_max：波反射
- **左右壁** x 边界：波反射
- **底部**：不涉及（水底不显示）

## 实现参数参考

| 参数 | 建议值 | 说明 |
|------|--------|------|
| X 分辨率 | 180~200 采样点 | 水面线平滑度 |
| Z 分辨率 | 30~50 采样点 | 够用不过深 |
| 阻尼 | 0.96~0.985 | 衰减速度 |
| 波速 | 0.1~0.2 | 传播速度 |
| 折射强度 | 与梯度成正比 | 由法线计算，非固定值 |
| 鼠标脉冲 | 0.3~0.7 | 涟漪强度 |
| 水体淡蓝 | rgba(150,200,235,0.15) | 极淡覆盖 |
| 水面线颜色 | rgba(0,0,0,0.3) | 深灰线 |
| 弹簧刚度 | 8 | 倾斜回弹 |
| 弹簧阻尼 | 3.5 | 倾斜衰减 |

## 帧循环

```
每帧:
  1. 波方程时间步 → 更新 h(x,z)
  2. 逐点计算梯度 (∂h/∂x, ∂h/∂z) → 法线
  3. 法线 → 折射偏转量
  4. 画上半屏（直接拷贝背景）
  5. 逐像素折射采样画下半屏
  6. 画水面线 h(x,0)
  7. 叠淡蓝水体
```
