# 彩蛋效果

## 🌊 水面狂暴

**触发**：2 秒内连续点击水面线 7 次

**效果**：水面剧烈晃荡 5 秒，随机脉冲不断注入波场，Z 倾斜来回震荡

**实现**：`WaterPool.tsx` — `onWindowClick` 检测点击位置是否在水面线 ±30px 内，`triggerRage()` 注入 20 个强涟漪脉冲并设置 5 秒狂暴窗口

---

## 🎆 Konami Code 烟花

**触发**：键盘输入 `↑ ↑ ↓ ↓ ← → ← → B A`

**效果**：全屏 300 颗彩色粒子从屏幕中心径向炸裂，带重力下落，逐渐消散

**实现**：`WaterPool.tsx` — `onKey` 匹配按键序列，`triggerKonami()` 生成 300 粒子数组，每帧更新位置和生命值

| 参数 | 值 |
|------|-----|
| 粒子数 | 300 |
| 半径 | 10px |
| 速度 | 3~21 px/frame |
| 颜色 | 7 色（indigo/紫/青/粉/黄/玫瑰/天蓝） |
| 生命周期 | ~67 帧（约 1 秒） |
| 重力 | 0.05 px/frame² |
