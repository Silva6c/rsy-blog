# 彩蛋效果

## 🌊 水面狂暴

**触发**：2 秒内连续点击水面线 7 次

**效果**：水面剧烈晃荡 5 秒，随机脉冲不断注入波场，Z 倾斜来回震荡

**实现**：`WaterPool.tsx` — `onWindowClick` 检测点击位置是否在水面线 ±30px 内，`triggerRage()` 注入 20 个强涟漪脉冲并设置 5 秒狂暴窗口

---

## 🎆 Konami Code 烟花

**触发**：键盘输入 `↑ ↑ ↓ ↓ ← → ← → B A`

**效果**：全屏 300 颗彩色粒子从屏幕中心径向炸裂，带重力下落，逐渐消散

**实现**：`WaterPool.tsx` — `onKey` 匹配按键序列，`triggerKonami()` 生成 300 粒子数组，每帧更新位置和生命值

| 参数 | 值 |
|------|-----|
| 粒子数 | 300 |
| 半径 | 10px |
| 速度 | 3~21 px/frame |
| 颜色 | 7 色（indigo/紫/青/粉/黄/玫瑰/天蓝） |
| 生命周期 | ~67 帧（约 1 秒） |
| 重力 | 0.05 px/frame² |

---

## ☄️ 陨石撞击

**触发**：背景图为 03_dark.png 时，点击水面以下任意位置

**效果**：一颗金色陨石从屏幕左上/右上拖尾飞来，撞击水面产生巨型冲击波 + 5 秒水面狂暴

**实现**：`WaterPool.tsx` — `onWindowClick2` 检测 bgIdx 和点击位置，`meteor` 对象跟踪陨石状态（位置/速度/拖尾/生死）

| 参数 | 值 |
|------|-----|
| 陨石半径 | 40px |
| 拖尾长度 | 30 点 |
| 主冲击波 | impulse 8 × spread 30 |
| 次级冲击波 | impulse 5 × spread 22 |
| 深层冲击波 | impulse 3 × spread 16 |
| 随机涟漪 | 20 个 |
| 狂暴时长 | 5 秒 |
